  <div class="sect1" id="starting_webaccess">
   <h1 class="title">
    Starting GroupWise WebAccess
   </h1>
   <div class="procedure">
    <ol class="steps">
     <li class="step" id="bq1r7b0">
      <p class="para">
       Use your Web browser to go to the URL your administrator gives you, such as
       <span class="literal">
        http://
        <span class="replaceable">
         server
        </span>
        /gw/webacc
       </span>
       , or an IP address such as 155.155.11.22.
      </p>
     </li>
     <li class="step" id="b6cdi48">
      <p class="para">
       On the GroupWise WebAccess login page, enter your user name and password.
      </p>
     </li>
     <li class="step" id="bwa1l46">
      <p class="para">
       Click either
       <span class="guimenu">
        This is a public or shared computer
       </span>
       or
       <span class="guimenu">
        This is a private computer
       </span>
       .
      </p>
      <p class="para">
       The option you choose affects the timeout interval that is used. To help keep your information secure, a shorter timeout interval is used if you are using WebAccess from a public or shared computer. For more information on timeout interval, see
       <a href="understanding_timing_out.html" class="xref">
        Understanding Timing Out from GroupWise WebAccess
       </a>
       .
      </p>
     </li>
     <li class="step" id="b6cdj9d">
      <p class="para">
       (Conditional) If you’re using a small mobile device such as a mobile phone, click
       <span class="guimenu">
        Options
       </span>
       , then select
       <span class="guimenu">
        Use the basic interface
       </span>
       .
      </p>
      <p class="para">
       The basic interface has limited functionality and graphics. It is designed to perform the most basic tasks in GroupWise WebAccess. To close the basic interface, close all Web browsers and launch WebAccess again, then deselect
       <span class="guimenu">
        Use the basic interface.
       </span>
      </p>
      <p class="para">
       For information on how to use the basic interface, see the
       <a href="http://www.novell.com/documentation/groupwise2012/gw2012_qs_webaccbasic/data/gw2012_qs_webaccbasic.html" class="ulink">
        GroupWise 2012 WebAccess Basic Quick Start
       </a>
       .
      </p>
     </li>
     <li class="step" id="b6cdkhr">
      <p class="para">
       Click
       <span class="guimenu">
        Login
       </span>
       .
      </p>
      <p class="para">
       Your GroupWise mailbox opens.
      </p>
      <p class="para">
       If you are unable to log in, click the
       <span class="guimenu">
        Can’t log in?
       </span>
       link on the login page for more information about how to get your password reset in your organization.
      </p>
     </li>
    </ol>
   </div>
