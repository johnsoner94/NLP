             </h3>
             <div class="has-user-generated-content js-d-read-more read-more js-read-more read-more--medium-down">
              <div class="js-xd-read-more-toggle-view read-more__toggle-view">
               <div class="js-xd-read-more-contents l-mar-top-3">
                <p>
                 Immediately following commencement ceremonies, graduates, families and guests are invited to attend an all-you-care-to-eat buffet luncheon in the Wood Dining Commons. The luncheon will feature Chicken Stir Fry, French Dip Sandwiches, Roasted Potatoes, Seasonal Vegetables, Variety of Hoagie-style Sandwiches, Cheese Pizza, Pepperoni Pizza, Salad Bar, Assorted Breads, Cookies, Brownies as well as Hot &amp; Cold Beverages. The Commencement Day Kosher menu will include: Pasta Primavera, Tuna Salad Sandwiches, Egg Salad Sandwiches, Cucumber Salad, Chocolate Chip Cookies.
                </p>
                <p>
                 Purchase your tickets by 5:00pm on Friday, April 15, 2016, to take advantage of the discounted rate of $13.99 for adults and $6.99 for children ages 5-12. Tickets purchased after 5:00pm on April 15, 2016 - May 20, 2016 will be available for $14.99 for adults and $7.99 for children ages 5-12.
                 <span>
                  Children under 5 eat for free.
                 </span>
                </p>
                <p>
                 <strong>
                  Deadline to order tickets is Friday, May 20 or until seating capacity is filled. Due to limited seating and the popularity of this event, we encourage you to purchase your tickets soon.
                 </strong>
                </p>
                <p>
                 <strong>
                  FAQs
                 </strong>
                </p>
                <p>
                 <strong>
                  Do I have to bring my printed ticket to the event?
                 </strong>
                </p>
                <p>
                 Yes. Tickets must be presented at the door to enter the Wood Dining Commons.
                </p>
                <p>
                 <strong>
                  The name on the ticket doesn't match the attendee. Is that okay?
                 </strong>
                </p>
                <p>
                 Yes.
                </p>
                <p>
                 <strong>
                  Can I purchase tickets at the door?
                 </strong>
                </p>
                <p>
                 Maybe. The Commencement Luncheon has proven to be very popular as it has sold out the past few years. While tickets were available at the door in 2015, we cannot guarantee they will be available for 2016.
                </p>
                <p>
                 <strong>
                  Is there reserved seating?
                 </strong>
                </p>
                <p>
                 No. Please be assured there will be plenty of seating, however, seating is based off of a first come, first served basis. Doors will not open until commencement ceremonies have concluded.
                </p>
                <p>
                 <strong>
                  Does graduating students need to purchase a ticket?
                 </strong>
                </p>
                <p>
                 Yes.
                </p>
                <p>
                 <strong>
                  What if I or someone in my party has a food allergy?
                 </strong>
                </p>
                <p>
                 While the Wood Dining Commons is not an allergen-free facility, our professional culinary team is trained to prevent cross-contamination. If you or someone in your party has a food allergen, please alert our staff.
                </p>
                <p>
                 <strong>
                  Who do I contact if I have a additional questions?
                 </strong>
                </p>
                <p>
                 Please e-mail us at Diningcomments@muhlenberg.edu. Please include "Commencement Lunch" in the subject line.
                </p>

   window.loggerSetup = function(settings, data) {
	'use strict';
	var logURL = settings.loggingEndpoint,
		maxMessagesPerHour = settings.maxMessagesPerHour,
		allowedURLs = settings.allowedURLs,
		allowedURLsCount = settings.allowedURLs.length,
		originalPageName = data.page,
		simpleConsole = window.console,
		browserStorage = window.localStorage,
		msInOneHour = 60 * 60 * 1000,
		Logger = window.Logger,
		pageOpenTime = (new Date()).getTime();
	if (!Logger) {
		return;
	}
	Logger.useDefaults({
		logLevel: Logger.WARN
	});
	function isAllowedToLogFromPage() {
		var i = 0;
		if (window.location.protocol.indexOf('https') === -1) {
			return false;
		}
		for (i = 0; i &lt; allowedURLsCount; i++) {
			if (new RegExp(allowedURLs[i]).test(document.location.pathname)) {
				return true;
			}
		}
		return false;
	}
	function appendDataFromCookie() {
		try {
			data.serverID = document.cookie.match(/SERVERID=?([a-zA-Z0-9]+)[ ]*;/)[1];
		} catch (e) {
			data.serverID = 'null';
		}
	}
	appendDataFromCookie();
	Logger.getOriginalPageName = function() {
		return originalPageName;
	};
	Logger.getPageName = function() {
		return data.page;
	};
	Logger.setPageName = function(name) {
		data.page = name;
	};
	Logger.appendDetails = function(key, value) {
		data.pageDetails[key] = value;
	};
	if (!settings.isLoggerEnabled || !isAllowedToLogFromPage()) { return; }
	function logIfPossible() {
		if (simpleConsole.log) {
			simpleConsole.log.call(simpleConsole, arguments);
		}
	}
	function isAllowedToLogMessages() {
		if (
			browserStorage &amp;&amp;
			parseInt(browserStorage.getItem('jsLogger:messageCount'), 10) &gt;= maxMessagesPerHour
		) {
			return false;
		}
		return true;
	}
	function dataFilter(data) {
		// These are really wide net filters. They will grab things they shouldn't.
		// It is better to over filter here. SSN also partially filters phone numbers
		// in some formats.
		var ccFilter = /(?:\d[ -]*?){13,16}/g,
			filterString = '[FILTERED]';
		return data
			.replace(ccFilter, filterString);
	}
	function setupLoggingRateLimiting() {
		var now = (new Date()).getTime(),
			nextHourMark;
		if (!browserStorage) { return; }
		nextHourMark = parseInt(browserStorage.getItem('jsLogger:nextHourMark'), 10);
		if (!nextHourMark || nextHourMark &lt; now) {
			browserStorage.setItem('jsLogger:nextHourMark', now + msInOneHour);
			browserStorage.setItem('jsLogger:messageCount', 0);
		}
	}
	function incrementMessagesLogged() {
		var nextMessageCount;
		if (!browserStorage) { return; }
		nextMessageCount = parseInt(browserStorage.getItem('jsLogger:messageCount'), 10);
		browserStorage.setItem('jsLogger:messageCount', nextMessageCount + 1);
	}
	function sendMessage(data) {
		var XHRKlass, logXHR;
		setupLoggingRateLimiting();
		if (!isAllowedToLogMessages()) { return; }
		logIfPossible('JSLogging: Transmitting', data);
		// send beacon works even while switching pages
		if (navigator.sendBeacon) {
			try {
				navigator.sendBeacon(logURL, data);
			} catch (e) {
				logIfPossible(e);
			}
		} else {
			try {
				// can't rely on jquery since it isn't loaded yet
				// should work IE5.5+
				XHRKlass = window.XMLHttpRequest || window.ActiveXObject,
				logXHR = new XHRKlass('MSXML2.XMLHTTP.3.0');
				logXHR.open('POST', logURL, true);
				logXHR.setRequestHeader('X-Requested-With', 'XMLHttpRequest');
				logXHR.setRequestHeader('Content-type', 'application/x-www-form-urlencoded');
				logXHR.send(data);
			} catch (e) {
				// if all else fails complain about it
				logIfPossible(e);
			}
		}
		incrementMessagesLogged();
	}
	Logger.setHandler(function (messages, context) {
		var stack = '',
			secondsSinceStartup,
			message;
		if (messages[4] &amp;&amp; messages[4].stack) {
			stack = messages[4].stack;
		}
		secondsSinceStartup = ((new Date()).getTime() - pageOpenTime) / 1000;
		message =
			'logLevel=' + encodeURIComponent(context.level.name) +
			'&amp;messages=' + encodeURIComponent(dataFilter(messages[0])) +
			'&amp;server_id=' + encodeURIComponent(data.serverID) +
			'&amp;page=' + encodeURIComponent(data.page) +
			'&amp;page_details=' + encodeURIComponent(JSON.stringify(data.pageDetails)) +
			'&amp;time_alive=' + encodeURIComponent(secondsSinceStartup) +
			'&amp;full_url=' + encodeURIComponent(data.fullURL) +
			'&amp;user_is_authenticated=' + encodeURIComponent(data.userIsAuthenticated) +
			'&amp;user_id=' + encodeURIComponent(data.userId) +
			'&amp;user_email=' + encodeURIComponent(data.userEmail) +
			// Stack is least important. If it gets cut off, we are ok with that.
			'&amp;stack=' + encodeURIComponent(dataFilter(stack))
		;
		sendMessage(message);
	});
	window.onerror = Logger.error;
};
